# レシピ管理アプリ

自分だけのレシピを管理・改善できるWebアプリです。

## 起動方法

コマンドプロンプトまたはターミナルで以下を実行:

```
cd C:\Users\kazum\OneDrive\デスクトップ\recipe-app
npm run dev
```

ブラウザで http://localhost:3000 を開く。

## 機能

### ホーム画面
- 登録済みレシピが写真付きグリッドで表示される
- 各レシピカードに推定コストとバージョン数が表示される
- クリックでレシピ詳細へ移動

### レシピ登録（+ 新規レシピ）
- 料理名、説明、写真、参考URLを入力
- 材料：食材名を入力すると自動でマスタから候補が出て、選択するとコスト自動計算
- 新しい食材は単価を入力するとマスタに自動登録される
- 作り方を手順ごとに入力

### レシピ詳細
- 材料一覧と各コスト、合計コストを表示
- 参考URLリンク
- 作り方の手順
- バージョン選択タブ（複数バージョンがある場合）
- 改善履歴タイムライン
- 2つのバージョンの差分比較（変更した材料を色付きで表示）

### 改善版を追加
- 「改善版を追加」ボタンから改善メモを入力して新バージョン作成
- 前バージョンの材料・手順がコピーされるので変更箇所だけ修正

### 食材マスタ
- 食材名、単位、単価を管理
- レシピ登録時の自動コスト計算に使われる
- 例: 1kgで¥200の小麦粉なら単価は 0.2（¥/g）

## データ保存場所

- レシピ・食材データ: `dev.db`（SQLiteデータベース）
- アップロード写真: `public/uploads/`

> ⚠️ `dev.db` を削除するとすべてのレシピデータが消えます。バックアップとして定期的にコピーしてください。
